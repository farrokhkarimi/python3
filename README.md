# - Exception Handling
```
try:
   statement(s)
   ......................
except:
   If there is any exception, then execute this statement(s)
   ......................
else:
   If there is no exception, then execute this statement(s)
finally:
   This would always be executed.
```

Ex:
```python
try:
    print('Start!')
    1 / 0
except:
    print('Error!')
else:
    print('Ok')
finally:
    print('Finish!')
```
Result:
```
Start!
Error!
Finish!
```
